このプラグインは、ツクールMZを極限までRPG MAKER WITHに近づけることを目的としたプラグインです。

競合対策など何も考えていませんが、どうぞご自由にお使いください。
※ステータス画面とショップ画面は力尽きたので適当です...ごめんなさい。


この中に入っているサンプルプロジェクトは、ツクールMZを持っていればすぐに遊ぶことが出来ます。


このプラグインセットが必要とする前提プラグインは、サンプルプロジェクトの中にあります。


サンプルプロジェクトの大まかな概要は次の通りです。

================================================================================
タイトル: 囚われのお姫様

勇者が助けに来ないから、さらわれた姫が魔王を倒す！
そんな感じの脱出アドベンチャー！

この作品は、RPG MAKER WITHの同名の作品と、同時並行で作成されました。

-----------------------------
あらすじ

ここは巨大樹「ゲルマニア」の上にある国「ミスト王国」
その国の姫である「姫騎士ミスト」はある日、異国へと視察に行くことになった。
しかし王国の外へ出ようとしたとたん、侵略者に襲われ...!?
